# Legacy Node.js backend (deprecated)

This folder is the **old Express + better-sqlite3** API. The site now uses **`backend-php/`** only (PHP + SQLite).

- **Database:** use `backend-php/database.sqlite` (your data was copied there).
- **Do not run** `npm run dev` here unless you are maintaining the old stack.

See [../backend-php/README.md](../backend-php/README.md) for setup and Afrihost deployment.
